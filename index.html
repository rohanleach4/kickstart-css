CSS Kickstarter for rapid prototyping

This is CSS Kickstarter uses HTML5 Boilerplate for it's structure.

The premise of this is to allow the developer to rapidly build prototypes, concentrating soley on semantic structure and not style classes. Initially, the only class that needs to be added to the html is the first tag: container

Remember, this isn't so much to build web apps, but protoypes.

The Kickstart css file an be found here: css/kickstarter.css
